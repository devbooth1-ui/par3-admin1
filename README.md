# admin1

## Setup

1. Install dependencies  
   `npm install`

2. Run in development  
   `npm run dev`

3. Build for production  
   `npm run build`

## Code Structure

- `components/` – React/Next.js components
- `hooks/` – Custom React hooks
- `lib/` – Shared library code
- `pages/` – Next.js pages and API routes
